Summary

Create white noise.

Audio Connections

PortPurpose
Out 0White Noise

Functions

amplitude(level);

Set the output peak level, from 0 (off) to 1.0. The default is off. Noise is generated only after setting to a non-zero level.

Examples

File > Examples > Audio >

Notes

Setting the amplitude to zero causes this object to stop using CPU time to generate random numbers.
